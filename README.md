# real-time-translation


## Run Locally

Clone the project

```bash
  git clone https://github.com/zablon-oigo/real-time-translation-api.git
```

Go to the project directory

```bash
  cd real-time-translation
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  fastapi dev main.py
```
